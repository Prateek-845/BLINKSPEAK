# BLINKSPEAK: Eye Controlled Communication System

An assistive technology project that translates eye blinks into Morse code and then into English text using Deep Learning.

---

###  Overview

This project provides a hands-free communication interface for individuals with motor impairments.  
By detecting eye blinks in real-time using a webcam, the system interprets the duration of blinks as Morse code "dots" and "dashes," translating them into text displayed on the screen.

The core of the system is a custom Convolutional Neural Network (CNN) trained on the MRL Eye Dataset to accurately classify eye states as **Open** or **Closed**.

---

###  Key Features

- **Deep Learning Core:** 11-layer custom CNN trained to ~99% accuracy on the MRL Eye Dataset.  
- **Robust Training:** End-to-end pipeline for data preprocessing, augmentation, model architecture search, and hyperparameter tuning.  
- **Explainable AI:** Grad-CAM (Gradient-weighted Class Activation Mapping) used to verify that the model focuses on meaningful eye regions instead of noise.  
- **Real-Time Potential:** Lightweight architecture suitable for real-time inference on standard hardware.  
- **Comprehensive Analysis:** Detailed metrics (Accuracy, AUC, Precision, Recall) and visualizations (ROC curves, confusion matrices).

###  Installation & Setup

#### Prerequisites

- Python 3.6 or higher  
- A webcam (for the upcoming real-time module)  
- GPU recommended for training (CPU is sufficient for inference)

#### Step 1: Clone the Repository

git clone https://github.com/Prateek-845/BLINKSPEAK.git
cd BLINKSPEAK

#### Step 2: Create a Virtual Environment

It is recommended to use a virtual environment to manage dependencies.

#### Windows
python -m venv venv
.\venv\Scripts\activate

#### Mac/Linux
python3 -m venv venv
source venv/bin/activate


#### Step 3: Install Dependencies

pip install -r requirements.txt

###  Usage Guide

#### 1. Data Preparation

The raw MRL Eye Dataset images must be processed first.

python preprocess.py


- **Input:** Reads raw images from `mrleyedataset/`.  
- **Output:** Creates `processed_dataset/` with resized (128×128), grayscale images split into train/val/test sets.

#### 2. Initial Model Architecture Search (Optional)

To search for the best base architecture (e.g., 10 vs. 15 vs. 20 layers):

python train_models.py

- **Output:** Generates `training_results/cnn_model_metrics.xlsx` comparing different CNN depths.

#### 3. Model Training & Hyperparameter Tuning

To train the chosen architecture and find the best hyperparameters (learning rate, dropout, etc.):

python tuning.py

- **Output:**  
  - Saves the best model as `tuning_results/best_hyper_model.pth`.  
  - Logs detailed metrics for all experiments in `tuning_results/tuning_metrics.xlsx`.

#### 4. Model Verification (Grad-CAM)

To visualize what the model is "looking at" to make its decision:

python gradcam_pytorch.py


- **Output:**  
  - Generates heatmap overlays in `tuning_results/gradcam_images/`, showing model attention on the eye region.

###  Performance

The final model was selected after a grid search over 64 hyperparameter combinations.

| Metric         | Score    |
|----------------|----------|
| Test Accuracy  | 98.99%   |
| Test AUC       | 0.9993   |
| Precision      | 98.66%   |
| Recall         | 99.37%   |
| Best Val Loss  | 0.0306   |

**Winning Configuration:**

- Architecture: 11-layer custom CNN  
- Learning Rate: 0.0001  
- Dropout: 0.4  
- Dense Units: 128  
- Batch Size: 64
